# siamese-vs-persian-cnn
Deep Learning project using TensorFlow to classify Siamese and Persian cats using Convolutional Neural Networks.

**Dataset:**This project uses the **Oxford-IIIT Pet Dataset**:
https://www.robots.ox.ac.uk/~vgg/data/pets/
 Dataset details:
- 37 different pet categories
- annotated images
- includes both Siamese and Persian cat breeds

For this project, only the following classes were used:
- Siamese
- Persian

Requirements

- Python (compatible with TensorFlow)
- Updated pip
- Venv
